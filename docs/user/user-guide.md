# Guide utilisateur SYNORA

## Objectif

SYNORA permet de connecter un wallet, consulter sa reputation Web3, reclamer des rewards SYN et utiliser un assistant IA.

## Prerequis

- MetaMask
- Reseau Base Sepolia
- Wallet connecte
- Quelques ETH testnet pour les transactions

## Connexion

1. Ouvrir le dashboard
2. Cliquer sur Connect wallet
3. Signer le message MetaMask
4. Attendre la restauration de session

## Reputation

Le score augmente selon les actions:

- wallet authentifie
- dashboard visite
- balance SYN connectee
- reward reclame

## Rewards

SYNORA permet:

- claim off-chain MVP
- claim on-chain via RewardsDistributor

## Leaderboard

La page Leaderboard affiche les wallets classes par score, rewards et activite.

## Badges

Les badges montrent les accomplissements utilisateur:

- Wallet Verified
- Early Adopter
- SYN Connected
- Reward Claimer
- On-chain Pioneer
- Top Reputation

## Assistant IA

Le bouton d'aide en bas a droite permet de poser des questions sur SYNORA.

## Important securite

Ne jamais partager:

- cle privee
- seed phrase
- phrase de recuperation
- acces MetaMask

L'equipe SYNORA ne demandera jamais ces informations.
