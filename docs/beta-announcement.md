# Annonce Beta SYNORA

## Version courte

SYNORA Beta est disponible sur Base Sepolia.

SYNORA combine authentification wallet, reputation Web3, rewards SYN on-chain, leaderboard, badges, analytics et assistant IA dans une experience utilisateur complete.

## Version longue

SYNORA entre en Beta testnet sur Base Sepolia.

Le projet propose une experience Web3 complete:

- connexion MetaMask
- authentification par signature
- score de reputation
- rewards SYN on-chain
- leaderboard
- badges utilisateur
- analytics
- assistant IA
- PWA installable
- interface FR / EN

Cette Beta permet de tester les parcours utilisateur, la reputation, les claims on-chain et les interactions avec l'assistant IA.
