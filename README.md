# SYNORA

SYNORA est une plateforme Web3 basée sur :

- Token ERC-20 SYN
- Authentification wallet
- Système de réputation
- Dashboard utilisateur
- Déploiement Base Sepolia
- Gouvernance future

## Structure

- apps/web : frontend Next.js
- apps/api : backend API
- contracts : smart contracts Hardhat
- packages/shared : types et logique partagée
- packages/config : configuration commune
- docs : documentation projet
- infra : configuration Render et Vercel
- scripts : scripts utilitaires
